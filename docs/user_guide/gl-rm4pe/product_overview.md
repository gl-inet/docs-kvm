## Touchscreen

| Screen Display                    | Description         |
| :-------------------------------- | :------------------ |
| ![booting](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/booting.png)| **Starting up**<br>Power on your Comet X. The GL.iNet logo appears, followed by a startup progress bar. Once the progress bar is fully loaded, the device startup is complete.|
| ![screen password](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/passcode.png)| **Set up screen password**<br>Set a screen password for your Comet X to protect your privacy.|
| ![home](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/home.png)| **Home screen**<br>**Top-right**: Cloud status and system time.<br>**Middle**: Device IP address, shown only when it connects to Internet.<br>**Bottom**: Four ports corresponding to four signal sourses. Blue indicates a physical connection, while a green checkmark indicates that this port is in use.<br>Click each port to view the HDMI signal, keyboard & mouse signal, and USB connected status. |
| ![switch signal](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/switch_signal.png)| **Switch Siganl Source**<br><u>Short press</u> the switch button on the front panel to switch signal source. Press the button again to switch and apply. |
| ![basic settings](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/basic_settings.png)| **Basic settings**<br>On the home screen, swipe left from the right to access basic settings, including **Screen Display** and **Cloud Service**.|
| ![system actions](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/system_menu.png)| **System Menu**<br>On the home screen, swipe down from the top to access system menu, including **Reboot** and **Lock Screen**. |
| ![time](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/time.png)| **Lock screen**<br>The lock screen displays the time and date. |
| ![reset](https://static.gl-inet.com/docs/kvm/user_guide/gl-rm4pe/touchscreen/reset.png)| **Factory Reset**<br>With the KVM powered on, <u>long press</u> the switch button on the front panel for 8 seconds to factory reset. Release the button within 8 seconds to cancel the reset. |
