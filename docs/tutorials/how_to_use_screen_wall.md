# How to use the cloud-based screen wall feature?

With the Screen Wall, you can view the screens of multiple devices on a single page and keep track of their statuses easily.

This tutorial will introduce how to use the cloud-based screen wall feature.

**Note:** Please upgrade your KVM firmware to version 1.7 before using this feature.

1. Bind your KVM device to your Cloud account, then log in to the Cloud. Click [here](http://127.0.0.1:8000/kvm/en/faq/remote_access_to_controlled_device_via_cloud/#1-bind-your-device-to-kvm-cloud) for detailed instructions.

    ![cloud login](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/cloud_login.png){class="glboxshadow"}

2. After login, you will see the bound devices in the device list. Make sure they are online.

    ![device list](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/device_list.png){class="glboxshadow"}

3. Click **Screen Wall** in the left side menu, and click **Setup Screen Wall**.

    ![screen wall](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/setup_screen_wall.png){class="glboxshadow"}

4. Select the KVM devices you want to view on the screen wall, and click **Save Screen Wall** in the upper right corner.

    ![select device](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/select_device.png){class="glboxshadow"}

    If your device is running firmware lower than version 1.7, it will be grayed-out and unselectable. Please upgrade the firmware first before using this feature.

    ![not supported firmware](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/not_supported.png){class="glboxshadow"}

5. After saving the settings, it will start connecting to the selected device over the Cloud.

    ![connecting](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/connecting.png){class="glboxshadow"}

6. Once connected, enter the admin password for each of your KVM devices.

    ![admin login](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/admin_login.png){class="glboxshadow"}

7. You will then be able to view the screens of multiple devices on a single page.

    ![screen wall display](https://static.gl-inet.com/docs/kvm/tutorials/how_to_use_screen_wall/screen_wall_display.png){class="glboxshadow"}
