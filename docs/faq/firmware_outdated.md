# How to upgrade firmware?

The cloud platform only supports firmware **v1.5.0 and above**. If your device is running an older version, you'll see the following message:

"Firmware outdated. Please upgrade to v1.5.0 or above via local or remote access."

![Firmware Outdated](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/firmware_outdated_.png){class="glboxshadow"}

## Local Access Upgrade 

1. [Local access](local_access_to_controlled_device_via_browser.md) to the KVM console. 

2. Click on the firmware version, then select "**Upgrade Now**".

    ![Firmware update](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/firmware_update1.png){class="glboxshadow"}

    ![Upgrade Now](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/upgrade_now_pic.png){class="glboxshadow"}

3. If you need to download the firmware manually, visit the [Firmware Center](https://dl.gl-inet.com/kvm){target="_blank"} and select the lastest firmware.

4. Then go back to the KVM console choose **Local Upgrade**, and wait for the installation to complete.

    ![Local Upgrade](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/local_update_pic.png){class="glboxshadow"}

## Remote Access Upgrade 

**Note**: Please make sure your device is already bound to your account.

1. Install the legacy GLKVM app [here](https://www.gl-inet.com/app-rm/?_gl=1#:~:text=Choose%20Your%20Platform){target="_blank"}.

     ![Download](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/app_download.png){class="glboxshadow"}

2. Log in with your account credentials.

    ![Login](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/Login.png){class="glboxshadow"}

3. Your bound device will appear — access it remotely and upgrade the firmware.

    ![Remote Control](https://static.gl-inet.com/docs/kvm/faq/firmware_outdated/remote_control.png){class="glboxshadow"}
